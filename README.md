 # Para Marce ❤️

Este es el diario de desarrollo de la página que estoy construyendo. Aquí anotaremos el progreso y los planes a futuro.

## Stack

* React
* TypeScript
* Vite

## Iniciar el proyecto en local

```bash
npm build
npm run dev
```

---

## Progreso Actual

*   **Configuración Inicial:** Proyecto creado con Vite, React y TypeScript.
*   **Estructura Base:** El componente principal `App.tsx` está funcionando.
*   **Estilos Globales:** Se ha configurado `index.css` para manejar estilos de toda la página, como el color de fondo y la tipografía base para `h1` y `p`.
*   **Layout Principal:** Se ha utilizado Flexbox en `App.css` para crear un layout que centra todo el contenido (vertical y horizontalmente) en la pantalla.
*   **Contenido Inicial:** Se ha añadido un título `<h1>` y un párrafo `<p>` con el primer mensaje.
